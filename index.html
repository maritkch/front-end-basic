Список основных команд Git

  1. git --version - показать версию git
  2. git init - создать новый репозиторий в текущеи каталоге
  3. git -status - показать статус новых документов в проекте
  4. git add . - добавить все новые файлы для дальнейшего коммита
  5. git clone - копирует репозиторий на локальный компьютер
  6. git commit -m "" - добавить файлы для коммита и закомментировать изменения в проекте
  7. git rm fileName.txt - удаляет файли для коммита, где fileName.txt имя удаляемого файла
  8. git remote -v - посмотреть где находится репозиторий
  9. git push origin main - отправить изменения, где main- нужная ветка или HEAD - для текущей ветки
  10. git branch - посмотреть сколько всего веток и какая текущая
  11. git checkout myBranch - переключиться на ветку, где myBranch - имя нужной ветки
  12. git checkout -b nameNewBranch - переключиться и создать новую ветку с именем nameNewBranch
  13. git commit -am "" - сделать коммит если не добавлялись новые файлы
  14. git log -p - подробно посмотреть историю всех коммитов
