Список основных команд Git

  1. git -version - показать версию git
  2. git -status - показать статус новых документов в проекте
  3. git add . - добавить файлы для коммита
  4. git commit -m "" - закомментировать изменения в проекте
  5. git remote -v - посмотреть где находится репозиторий
  6. git push origin main - отправить изменения, где main- нужная ветка или HEAD - для текущей ветки
  7. git branch - посмотреть сколько всего веток и какая текущая
  8. git checkout myBranch - переключиться на ветку, где myBranch - имя нужной ветки
  9. git checkout -b nameNewBranch - переключиться и создать новую ветку с именем nameNewBranch
